# Markr.exe

A minimal image watermarking utility built with Flask.

Markr.exe lets you quickly add text watermarks to images with customizable position and opacity. The tool provides a simple interface with a live preview to help you place your watermark before generating the final image.

## Features

* Upload an image
* Add a custom watermark text
* Choose watermark position
* Adjust watermark opacity
* Preview watermark placement
* Download the final watermarked image

## Tech Stack

* Python
* Flask
* HTML / CSS
* JavaScript
